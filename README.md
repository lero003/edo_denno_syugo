# 江戸 × AI | 電脳浮世絵巻 - Ultimate

> 「歴史は繰り返さないが、韻を踏む。」

## 概要 (Overview)
本プロジェクトは、江戸時代の精神性（OS）と現代のAI技術（Hardware）を習合させた、実験的ウェブサイトである。
「わび・さび」を最適化アルゴリズムとして再解釈し、浮世絵のレイヤー構造をニューラルネットワークに見立てることで、過去と未来の特異点（シンギュラリティ）を描き出す。

## 機能 (Features)

### 01_REDEFINITION (温故知新)
江戸の循環型社会とAIの学習プロセスの類似性を提示。北斎がGPUを手にしていたら？という歴史のifを実装。

### 02_CHRONOLOGY (時層年表)
1603年の幕府開府（OSインストール）から、2045年のシンギュラリティ（令和の元禄文化）までのタイムライン。
- **黒船来航**: DDoS攻撃として定義
- **明治維新**: OSメジャーアップデート

### 03_ALGORITHM (作法と論理)
伝統的価値観をコードとして記述。
- **TEA_CEREMONY.py**: 茶道のプロトコル
- **BUSHIDO.java**: 武士道のエラーハンドリング
- **KABUKI.css**: 歌舞伎のUI/UX
- **HAIKU.zip**: 俳句の圧縮アルゴリズム

### 04_CHAT_LOG (時空対話記録)
葛飾北斎、松尾芭蕉、平賀源内といった江戸のクリエイターと、現代AIのチャットログ。

### 05_LEXICON (電脳江戸辞典)
江戸用語をテック用語で翻訳。
- **飛脚**: パケット通信
- **鎖国**: ファイアウォール
- **切腹**: Force Quit

### 06_LATENT_SPACE (電脳浮世絵)
CSSとSVGで描画された、概念的な浮世絵ギャラリー。

### 08_ARCHITECTURE (城郭構造)
江戸城の防御システムをゼロトラストセキュリティとして解説。

### 09_TOKENOMICS (米本位制)
米（Koku）を基軸通貨とするトークンエコノミーの考察。

### 10_SKINS (装いと実装)
十二単（z-index）や家紋（QRコード）など、ファッションと実装の対比。

## インストール (Installation)

```bash
# リポジトリをクローン（分霊）
git clone https://github.com/your-username/edo-denno-syugo.git

# ディレクトリへ移動（参勤交代）
cd edo-denno-syugo

# ローカルサーバーを起動（開国）
python3 -m http.server 8080
```

## 使用方法 (Usage)
ブラウザで `http://localhost:8080` にアクセスし、スクロール（絵巻物を紐解く動作）してください。
